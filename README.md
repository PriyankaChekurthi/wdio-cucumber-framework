# WDIO + TypeScript + Cucumber Automation Framework

Sample automation framework built using:
- WebdriverIO (WDIO)
- TypeScript
- Cucumber (BDD)
- Page Object Model (POM)

## Features
- Reusable page objects
- Cucumber feature files
- API utility example
- Smoke test sample
- Scalable structure

## Run Project

```bash
npm install
npx wdio run wdio.conf.ts
```

## Folder Structure

- features/
- pages/
- utils/
- step-definitions/
